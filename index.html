ShiftCheckbox jQuery plugin

This jQuery plugin handles the following checkbox-related functions:

  • Passing clicks from a "row" element that contains a checkbox to the checkbox itself
  • Setting up a "Select all" checkbox and handling its events
  • Handling Shift+click by selecting/deselecting ranges of checkboxes (like GMail)
A couple demos of the plugin's functionality are below.

Demo 1 - All features in use

Click (or Shift+click) on any part of a row to select the corresponding checkbox. The rows are the div.checkbox elements, and the presence of the checkboxSelector option tells the plugin that it should handle click events on the rows and pass them to the checkboxes. Note that if a row contains clickable elements such as links or buttons, they should have a click handler which returns false if you do not want the clicks to be passed through to the checkboxes.

Code: 

$('#demo1 div.checkbox').shiftcheckbox({
  checkboxSelector: ':checkbox',
  selectAll: $('#demo1 .all')
});
To disable the plugin: $('#demo1 div.checkbox, #demo1 .all').unbind('.shiftcheckbox');

Demo 2 - Basic functionality

For this demo, only the checkboxes pick up the Shift+click events and not the labels attached to them. For usability purposes, it is better to wrap both the label and the checkbox in a row element and specify the checkboxSelector option like the first demo does, or leave out the labels entirely.

Code: $('#demo2 :checkbox').shiftcheckbox();
To disable the plugin: $('#demo2 :checkbox').unbind('.shiftcheckbox');
